## 🎵 Song Popularity Prediction
**DSC 148 · Machine Learning · Python**

Predicting the popularity of songs from a Spotify dataset using multiple regression models. The project evaluates **LightGBM**, **Random Forest Regressor**, and **XGBoost Regressor** against a Linear Regression baseline, with a focus on both predictive accuracy and model interpretability.

- Data cleaning and preprocessing on a real-world Spotify dataset
- Exploratory data analysis (EDA) and baseline modeling
- Feature importance analysis across all models to understand what drives popularity ratings and how those features compare across approaches

**Skills:** LightGBM, XGBoost, scikit-learn, pandas, data cleaning, feature importance

## 🏥 Multi-Dataset Medical & Wine Classification
**COGS 118A · Classification · Python**

A comparative study of three classification models — **Logistic Regression**, **K-Nearest Neighbors (KNN)**, and **Support Vector Machine (SVM)** — across three public UCI datasets:

- **Cervical Cancer** — predicting cancer diagnosis from patient health characteristics
- **Breast Cancer** — classifying tumors as malignant or benign
- **Wine** — classifying wine type from chemical attributes

Each model was evaluated across three train/test splits (20/80, 50/50, 80/20) with 3 trials per split, yielding 27 test error measurements per dataset. The goal was to identify the best-performing model per dataset and overall.

**Skills:** scikit-learn, logistic regression, KNN, SVM, model evaluation, UCI datasets

## ⚡ Power Outages Data Analysis & Prediction
**DSC 80 · Data Science · Python** · *with Vinuthna Hasthi*

🔗 [View Project Website](https://vhasthi.github.io/power.outages-project/)

Analyzed 1,534 U.S. power outages to identify trends across climate regions, causes, and customer impact. Performed missingness analysis, hypothesis testing, and built a Random Forest regression model to predict customers affected — reducing MAE from 145,663 (baseline) to 87,811 through feature engineering and GridSearchCV tuning.

**Skills:** pandas, scikit-learn, RandomForestRegressor, GridSearchCV, permutation testing, data imputation
